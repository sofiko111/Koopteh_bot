Основная цель проекта — снять нагрузку с приемной комиссии и административного персонала, отвечая на наиболее часто задаваемые вопросы. Это позволяет сократить время ожидания ответов, повысить удовлетворённость пользователей и предоставить им актуальные данные без необходимости посещения веб-сайта или звонков. Бот использует простой, но эффективный подход на основе кнопок-подсказок, что делает его интуитивно понятным даже для пользователей, не имеющих опыта взаимодействия с чат-ботами.
